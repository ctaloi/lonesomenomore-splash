# Lonesome No More - Intake Question Framework

## Overview

This document outlines the comprehensive question framework for gathering personalization data about loved ones. This data powers the AI companion to create natural, meaningful conversations.

**Design Principles**:

- ✅ Structured yet flexible
- ✅ Easy for families to complete
- ✅ Actionable for AI conversation generation
- ✅ Respectful of privacy and sensitivity
- ✅ Progressive disclosure (quick start + optional depth)
